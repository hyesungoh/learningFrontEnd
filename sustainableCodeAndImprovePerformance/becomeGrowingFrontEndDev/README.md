## 옛날의 Web

옛날의 웹 개발자는 Frontend, Backend, 운영을 같이 하는 사람을 말했으나,

그 이후 AJAX 기술을 사용한 Web 2.0이 활발히 사용되었다.

2004년 AJAX로 개발된 Gmail이 서비스된 후, 큰 파장을 일으켜 국내외에 도입되기 시작하였다.
